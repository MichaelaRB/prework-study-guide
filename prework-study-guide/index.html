HTML

  • The head element contains information about the webpage.
  • The metadata contained in the head element is not visible on the webpage.
  • The !DOCTYPE element tells the browser what type of document it should expect. !DOCTYPE html tells the webpage to expect an HTML document.
  • The html element represents the start and end of our document.The opening tag defines the language the browser will use. For example, the opening tag for an english browser would read html lang=en.
  • The title element defines the title for the webpage.
  • The style element defines associated CSS styles.
  • The base element defines the webpage's base URL, and the link element connects an external resource.
  • The meta element defines metadata, such as the available character set.
  • The script element embeds client-side scripts.
  • The body element represents the visible content shown to the user.
  • The id attribute is used as a way to identify and select elements.
  • The ul element creates an unordered list, like this one! The ol element creates a numbered list.
  • For hyperlinks, the a element is used to create a linke, and the href attribute points to the URL.
  • The img element is used to add images. The src attribute points to the location of the image, alt is a string of text that describes the image, and you can set the width and height.
  • The p element represents a block of text.
  • The h1, h2, h3, h4, h5, and h6 elements represent different levels of headings. The h1 element is the largest heading, and h6 is the smallest.
